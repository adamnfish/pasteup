Pasteup is where design meets development. It is where the Guardian’s globally recognised design language is turned into code for the web, and the starting point when styling Guardian branded products for both internal teams and third parties.

Status

Pasteup is work in progress. The base CSS and layout CSS are ready for use and already power the page you are currently reading.

Further design components are being added over time. You can see modules that have been implemented in the module library.

Getting started

Include Pasteup in any HTML document and you should get some standard Guardian design features. This is the first and simplest way of using Pasteup.

  1. Add a link in the head of your HTML page to http://pasteup.gu.com/css/pasteup.min.css

Base CSS

The base CSS of Pasteup should give you a generic Guardian style design for your page. This includes standard fonts and font sizes, colors, and heading styles. This is the simplest way of using Pasteup, and should go some distance to giving you a Guardian look and feel.

You can see an example of how this might look in our styleguide.

Layout styles

grid.html

Guardian web pages typically implement one of two grid systems. A 940px fixed width grid as on guardian.co.uk, and a liquid and responsive layout which works across devices from mobile and tablet up to desktop (this doesn't exist yet).

Module library

modules.html

A set of stand-alone components which make up Guardian web pages. These should be entirely decoupled from any particular page or any particular layout.

Hosting

Pasteup is fully hosted on http://pasteup.gu.com which is a globally distributed and load balanced CDN. All files are minified, gzipped, and have far future Expires headers so they are aggressively cached by browsers.

In the first instance, please link to URLs on this domain, as they are likely to be already cached by your users.
