Pasteup | Guardian News & Media

Pasteup is where design meets development. It is where the Guardian's globally recognised design language is turned into code, and the starting point for styling Guardian branded web products both internally and for partners.

HTML Styleguide

base.html

Include Pasteup in any HTML document and you should get some good standard Guardian design features. This is the first and simplest way of using Pasteup.

Layout styles

grid.html

Guardian web pages typically implement one of two main grids. A 940px fixed width grid, and a liquid and responsive layout which works across devices from mobile and tablet up to desktop (this doesn't exist yet).

Module library

modules.html

A set of stand-alone components which make up Guardian web pages. These should be entirely decoupled from any particular page or any particular layout.

Hosting

Pasteup is fully hosted on http://pasteup.gu.com which is a globally distributed and load balanced CDN. All files are minified, gzipped, and have far future Expires headers so they are aggressively cached by browsers.

In the first instance, please link to URLs on this domain, as they are likely to be already cached by your users.
