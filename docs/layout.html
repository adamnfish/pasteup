This is created by the use of two class values, row and col-[n]. Here's the code that implements the final two rows of the above layouts.

<div class="row">
    <div class="col-8">8</div>
    <div class="col-4">4</div>
</div>
<div class="row">
    <div class="col-6">6</div>
    <div class="col-6">6</div>
</div>
