This is created by the use of two class values, row and col-[n]. Here's the code that implements the final two rows of the above layouts.

<div class="row">
    <div class="col-8">8</div>
    <div class="col-4">4</div>
</div>
<div class="row">
    <div class="col-6">6</div>
    <div class="col-6">6</div>
</div>

940px max width liquid grid

The liquid grid is implemented in percentages so that the widths of the columns scale along with the width of the users browser. Combined with media queries and flexible width images this can be used to implement a responsive design, which assumes less about your users browsing environment.
